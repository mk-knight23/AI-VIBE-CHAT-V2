# AI-VIBE-CHAT-V2: UI/UX Rebuild Plan

## Theme: "Minimalist Light with Neon Accents"

### Visual Identity

- Clean white background
- Minimal visual noise
- Neon cyan/purple accents for actions
- Generous whitespace
- Sharp edges (minimal rounding)
- Monospace for code, Inter for UI

### Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| --bg-primary | #ffffff | Main background |
| --bg-secondary | #f8fafc | Sidebar, cards |
| --bg-tertiary | #f1f5f9 | Input areas |
| --accent-cyan | #00d9ff | Primary actions |
| --accent-purple | #b829f7 | Secondary highlights |
| --text-primary | #0f172a | Main text |
| --text-secondary | #64748b | Muted text |
| --border | #e2e8f0 | Borders |

### Layout

- Clean sidebar (240px)
- Centered chat area (max-width: 900px)
- Floating input bar
- Minimal header
- Thread-based navigation

### Chat Bubbles

- User: White with cyan left border
- AI: Light gray background
- Sharp corners (4px radius)
- Clean typography
- Syntax highlighted code blocks
